Thank You for installing NateWorks Human sim
To install please follow below steps:
Note: This is only for use on linux operating systems as of now
1. run in terminal: git clone https://github.com/NatoTatoSHC/human.git
2. run: cd human
3. run: chmod +x install.sh
then run: ./install.sh
or run: bash install.sh
4. All ready, run: human, to begin messing around
